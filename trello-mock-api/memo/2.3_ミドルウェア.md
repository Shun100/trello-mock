# ミドルウェア

- リクエストがルートハンドラに届く前や、レスポンスがクライアントに返される前に通過する処理を挿し込むことができる
  - この処理のことをミドルウェアと呼ぶ

## ミドルウェアの基本構造

- ミドルウェア関数は3つの引数を受け取る
  - `req`: リクエストオブジェクト
  - `res`: レスポンスオブジェクト
  - `next`: 次のミドルウェア関数やルートハンドラへ処理を引き継ぐための関数

```TypeScript
  function middleWare(req, res, next) {
    req.customProperty = 'myCustomProperty'; // 例 リクエストオブジェクトに情報を追加
    next(); // 次のミドルウェアに情報を渡す

    console.log('リクエスト処理完了'); // 注意: next()を呼び出した後の処理も実行される
  }

  app.use(middleWare);
  app.use('/api', middleware);
```

## ミドルウェアの実行順序

- ミドルウェアは定義された順序で実行される

```TypeScript
  // 1つ目のミドルウェア
  app.use((req, res, next) => {
    console.log('First');
    next();
  });

  // 2つ目のミドルウェア
  app.use((req, res, next) => {
    console.log('Second');
    next();
  });

  // 3つ目のミドルウェア
  app.use((req, res, next) => {
    console.log('third');
    next();
  });

  // ルートハンドラ
  app.get('/', (req, res, next) => {
    res.send('Hello World');
  });

  // First
  // Second
  // Third
  // Hello World
```

## 組み込みミドルウェア

- Express.jsには、よく使われる機能を提供する組み込みミドルウェアがある

```TypeScript
  // 静的ファイル配信
  app.use(express.static('public'));

  // 特定のパスにマウント
  app.use('/assets', express.static('public'));

  // JSONリクエストボディの解析
  app.use(express.json());

  // URLエンコードされたフォームデータの解析
  app.use(express.urlencoded({ extended: true }));
```

### サードパーティミドルウェア

- 代表的なサードパーティ製ミドルウェアを紹介する

```TypeScript
  // morgan: HTTPリクエストロギング
  app.use(morgan('dev'));

  // cors: CORSサポート
  app.use(cors()); // デフォルト設定ではすべてのオリジンからのリクエストを許可する

  // helmet: セキュリティ関連のHTTPヘッダ設定
  app.use(helmet());

  // cookieParser: クッキー解析
  app.use(cookieParser());

  // multer: ファイルアップロード機能
  const upload = multer({ dest: 'uploads/' });
  app.post('/profile', upload.single('avatar'), (req, res) => {
    console.log(req.file); // アップロードされたファイル情報
    res.send('アップロード成功');
  })
```

### カスタムミドルウェア

- もちろん、自身でミドルウェアを作成することも可能

```TypeScript
  // リクエスト時間記録ミドルウェア
  function requestTime(req, res, next) {
    req.requestTime = Date.now();
    next();
  }

  app.use(requestTime);

  app.get('/', (req, res) => {
    res.send(`リクエスト時間: ${req.requestTime}`);
  });
```
